Table of Progress.js

A jquery plugin for making a table of contents that keeps track of your scrolling progress.

It's available on GitHub.

How Does it Work?

Just tell the plugin what element should be used as the table of contents container, and give it a jquery selector that describes the elements that you want to keep track of. toprogress.js will handle the rest for you.
